Number Guessing Game
A terminal-based guessing game written in Python. The program picks a random number between 1 and 10 and gives you unlimited guesses until you get it right.
How to Run
bashpython app.py
How It Works

A random number between 1 and 10 is generated
You're prompted to guess until you match it
Invalid input and out-of-range numbers are handled gracefully
Your total guess count is displayed when you win

What I Learned

Using a single while loop to handle all program states
Input validation with try/except
ANSI color codes for terminal output
